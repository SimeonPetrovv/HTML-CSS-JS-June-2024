While-цикъл

Основни принципи на while цикъл

Използване на while цикъл за повторение на код, докато дадено условие е вярно.

Примерна задача:

let i = 0;
while (i < 5) {
    console.log("Стойност на i: " + i);
    i++;
}

Do-while цикъл

Използване на do-while цикъл, който първо изпълнява кода, след което проверява условието.

Примерна задача:

let i = 0;
do {
    console.log("Стойност на i: " + i);
    i++;
} while (i < 5);

Промяна на променлива в цикъл

Промяна на стойността на променлива вътре в цикъл.

Примерна задача:

let i = 0;
while (i < 3) {
    console.log("Стойност на i: " + i);
    i++;
    if (i === 2) {
        i = 5; // Промяна на стойността на i
    }
}
