По-сложни проверки

Switch-case конструкция

Използване на switch-case конструкция за проверка на различни стойности на променлива.

Примерна задача:

            
                let day = 2;
                switch (day) {
                case 1:
                console.log("Понеделник");
                break;
                case 2:
                console.log("Вторник");
                break;
                case 3:
                console.log("Сряда");
                break;
                default:
                console.log("Друг ден");
                }

Вложени проверки

Използването на вложени if-else конструкции за проверка на условия в рамките на други условия.

Примерна задача:

            
                let number = 15;
                if (number > 0) {
                    if (number % 2 === 0) {
                    console.log("Числото е положително и четно.");
                    } else {
                    console.log("Числото е положително и нечетно.");
                    }
                } else if (number < 0) {
                console.log("Числото е отрицателно.");
                } else {
                console.log("Числото е нула.");
                }

Логически опратори: и, или, отрицание

Използването на логически оператори (&&, ||, !) за създаване на сложни условия.

Примерна задача:

let age = 25;
let isMember = true;

if (age >= 18 && isMember) {
    console.log("Достъп разрешен.");
} else {
    console.log("Достъп забранен.");
}
