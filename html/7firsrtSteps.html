Първи стъпки в JavaScript

Как работи JavaScript?

JavaScript е език за програмиране, използван за създаване на динамични и интерактивни уеб страници. Той се изпълнява на страната на клиента (в браузъра) и може да манипулира съдържанието на HTML страниците в реално време.

Примерна задача:

            
                <!DOCTYPE html>
                <html>
                <body>

                <p id="demo">Този текст ще бъде променен.</p>

                <script>
                document.getElementById("demo").innerHTML = "Hello, World!";
                </script>

                </body>
                </html>

Променливи

Променливите в JavaScript се използват за съхранение на данни, които могат да бъдат използвани и манипулирани по-късно в програмата.

Примерна задача:

            
                let x = 5;
                let y = 10;
                console.log(x + y); // Резултат: 15

Типове данни

JavaScript поддържа различни типове данни като числа, низове (текст), булеви стойности и др.

Примерна задача:

            
                let name = "John";
                let age = 25;
                console.log("Name: " + name + ", Age: " + age);

Вход/изход на конзолата

JavaScript може да приема вход от потребителя и да отпечатва изход в конзолата. 

            
                function () {
                    console.log("Hello, World");
                }

                function()

Примерна задача:

            
                let userName = prompt("Как се казвате?");
                console.log("Здравей, " + userName + "!");

Прости операции и пресмятания

JavaScript поддържа основни аритметични операции като събиране, изваждане, умножение и деление.

Примерна задача:


        let a = 8;
        let b = 4;
        console.log("Събиране: " + (a + b)); // 12
        console.log("Изваждане: " + (a - b)); // 4
        console.log("Умножение: " + (a * b)); // 32
        console.log("Деление: " + (a / b)); // 2
