Проверки

Оператори за сравнение

Операторите за сравнение се използват за сравнение на стойности и връщат булева стойност (true или false).

Примерна задача:

let x = 10;
let y = 20;
console.log(x < y); // true
console.log(x > y); // false
console.log(x == y); // false

Булева променлива

Булевите променливи могат да приемат само две стойности: true или false.

Примерна задача:

let isJavaScriptFun = true;
console.log(isJavaScriptFun); // true

Прости логически проверки

Използването на if-else условни конструкции за изпълнение на различни блокове от код в зависимост от логическо условие.

Примерна задача:

let number = 15;
if (number > 10) {
    console.log("Числото е по-голямо от 10.");
} else {
    console.log("Числото е по-малко или равно на 10.");
}

Серия от проверки

Използването на множество if-else if-else конструкции за проверка на различни условия.

Примерна задача:

let number = 0;
if (number > 0) {
    console.log("Числото е положително.");
} else if (number < 0) {
    console.log("Числото е отрицателно.");
} else {
    console.log("Числото е нула.");
}

Закръгляне и форматиране

Използването на вградени методи за закръгляне на числа.

Примерна задача:

let num = 4.7;
let roundedNum = Math.round(num);
console.log(roundedNum); // 5

Живот на променлива

Разликата между локални и глобални променливи и обхвата им.

Примерна задача:

let globalVar = "Глобална променлива";

function myFunction() {
    let localVar = "Локална променлива";
    console.log(globalVar); // Глобална променлива
    console.log(localVar); // Локална променлива
}

myFunction();
console.log(globalVar); // Глобална променлива
console.log(localVar); // Грешка: localVar is not defined

Дебъгване

Използване на console.log и други инструменти за дебъгване на JavaScript код.

Примерна задача:

let x = 5;
console.log("Стойност преди промяна: " + x);
x = 10;
console.log("Стойност след промяна: " + x);
