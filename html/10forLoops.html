For-цикъл

Инкрементация и декрементация

Използването на операторите ++ и -- за увеличаване или намаляване на стойността на променлива.

Примерна задача:

for (let i = 0; i < 5; i++) {
    console.log("Стойност на i: " + i);
}

Прости цикли

Използването на for цикъл за повторение на код определен брой пъти.

Примерна задача:

for (let i = 0; i <= 3; i++) {
    console.log("Брой: " + i);
}

Вложени цикли

Използването на цикли вътре в други цикли за обхождане на многомерни данни.

Примерна задача:

for (let i = 0; i < 3; i++) {
    for (let j = 0; j < 2; j++) {
        console.log("i = " + i + ", j = " + j);
    }
}

Цикли с масиви

Използването на for цикъл за обхождане на всички елементи на масив.

Примерна задача:

let colors = ["Red", "Green", "Blue"];
for (let i = 0; i < colors.length; i++) {
    console.log(colors[i]);
}

Цикли с обекти

Използването на for-in цикъл за обхождане на свойствата на обект.

Примерна задача:

let person = {
    name: "John",
    age: 30,
    city: "New York"
};

for (let key in person) {
    console.log(key + ": " + person[key]);
}

Цикли със символни низове

Използването на for цикъл за обхождане на символите в низ.

Примерна задача:

let str = "Hello";
for (let i = 0; i < str.length; i++) {
    console.log(str[i]);
}
